dk restrictions, etc. etc.

----------------------------------------------
-- INSTALLATION: -----------------------------
----------------------------------------------
Copy all 'bankeagle.php' into your modules folder.
Copy 'eagle.gif' into your images folder.

Login to the Superuser Grotto and Install / Activate it.
Uninstall / Deactivate 'instabank'.
